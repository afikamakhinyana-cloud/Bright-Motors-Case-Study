### 2. Historical_vs_Recent.csv

**Description:** This file contains the results of 11 SQL queries comparing older and recent years to identify trends, growth patterns and declining performance areas.

**Columns included:**
- sale_year
- units_sold
- total_revenue
- avg_selling_price
- make
- model
- era
- body_type
- avg_margin_pct
- state
- avg_odometer
- seller
- avg_condition_score
- sale_month
- sale_quarter
- month_name
- quarter_name

**Use this file for:** Section 2 of your Looker Studio dashboard (Historical Analysis — Blue charts)
